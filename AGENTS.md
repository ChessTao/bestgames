# Структура Проекта

## Общее Описание

Этот репозиторий представляет собой небольшой фронтенд-проект с плоской
структурой без вложенных исходных директорий. Основное приложение — HTML-страница
для просмотра и анализа шахматных партий, использующая один CSS-файл и один
JavaScript-файл.

## Файлы В Корне

- `Best Games Last.html`
  Основная текущая точка входа, открытая в редакторе. Подключает `Styles.css` и
  `App.js`, загружает `chess.js` через CDN и содержит основную разметку шахматного
  просмотрщика. В файле сейчас заметны проблемы с кодировкой текста и некорректный
  хвост документа: внизу повторно встречается `</head>`.

- `App.js`
  Главная логика приложения. Содержит встроенные PGN-данные в `RAW_PGN`,
  управление состоянием доски, навигацию по ходам, обработку анализа и
  вариантов, а также привязку логики к DOM-элементам интерфейса.

- `Styles.css`
  Основной файл стилей для текущей HTML-страницы. Определяет палитру, сетку
  макета, размеры доски, оформление панелей, кнопок и адаптивных элементов.

- `best games.html`
  Более старая или альтернативная HTML-версия страницы.

- `best games_fixed_clicks_and_persistence.html`
  Альтернативная HTML-версия, судя по названию, с правками кликов и сохранения
  состояния.

- `best games_with_variations.html`
  Альтернативная HTML-версия с акцентом на поддержку вариантов.

- `best games.pgn`
  PGN-файл с шахматными партиями. Вероятный исходный источник данных для
  просмотрщика.

- `best games.ini`
  Небольшой конфигурационный или служебный файл, связанный с проектом или
  экспортом данных.

## Рабочие Заметки

- Сейчас проект состоит из одного каталога, поэтому его легко просматривать
  вручную.
- Наиболее вероятные файлы для активной разработки:
  `Best Games Last.html`, `App.js`, `Styles.css`.
- В проекте есть несколько HTML-снимков или вариантов страницы, поэтому изменения
  стоит вносить осознанно в активный файл, а не механически копировать во все
  HTML-версии.
